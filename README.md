### `ACTIVAR EN TERMUX ` 
- ESCRIBE LOS SIGUIENTES COMANDOS UNO POR UNO:
```bash
> cd
> termux-setup-storage
> apt update 
> pkg upgrade 
> pkg install git -y
> pkg install nodejs -y
> pkg install ffmpeg -y
> pkg install imagemagick -y
> pkg install yarn
> git clone https://github.com/BrunoSobrino/Mystic-termux
> cd toby-bot
> npm install 
> yarn install 
> npm install
> npm update
> npm start
```
